## 🚀 Getting Started
1. Clone the repo:
    ```bash
    git clone https://github.com/your-username/sergio-portfolio.git
    cd sergio-portfolio
    ```

2.	Install dependencies:
    ```bash
    npm install
    ```
3.	Run locally:
    ```bash
    npm run dev
    ```
4.	Open http://localhost:3000
